Mastermind en python (Texte)

Introduction :


Demander lors d’un TP de NSI,Ce mini-jeu créé en 1 jour,reprend le principe du jeu de société MasterMind.


Présentation :


Ce jeu reprend le but du MasterMind, je l’adapte ici avec les couleurs remplacé par des nombres.Le nombre a trouvé est générer aléatoirement par l’ordinateur.Le but est que le joueur (humain) trouve la combinaison de chiffre.Pour cela l’ordinateur lui dira le nombre de couleur bien ou mal placés.
Ce jeu nécessite un IDE comme Pycharm, Visual Studio ou python portable pour lancer le fichier.
